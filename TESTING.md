## 🔍 문제 해결

### Q. 여전히 1MB 제한이 보입니다

**A.** 다음을 확인하세요:

1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 앱 재시작 (npm run dev)
3. 그래도 문제면 Supabase Storage 정책 확인

### Q. 파일은 선택되지만 업로드가 실패합니다

**A.** 다음을 확인하세요:

1. 네트워크 연결 확인
2. 브라우저 콘솔(F12)에서 에러 메시지 확인
3. 서버 로그 확인

### Q. 이미지 형식 오류가 발생합니다

**A.** 지원 형식(JPG, PNG, WebP)인지 확인하세요.

## 📝 결론

✅ **현재 모든 파일 크기 제한이 5MB로 통일되어 있습니다.**

사용자가 경험한 1MB 제한은:

- Supabase Storage의 이전 기본 설정
- 또는 과거 버전의 코드였을 가능성

현재 코드는 명확히 5MB 제한을 적용하고 있습니다.
